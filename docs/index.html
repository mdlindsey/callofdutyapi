Developers

Easily pull real-time data for team websites, Discord bots, or any other application.

Gamers

Contributors

Actively maintained and focused on reducing the effort required to complete your first PR.

Use data anywhere with no learning curve

In addition to offering the most comprehensive, up-to-date documentation on the Call of Duty API, we've meticulously mapped all request and response parameters to offer a developer experience easy enough for newcomers while maintaining the maturity expected for enterprise-scale applications.

Lightweight cross-platform libraries complete with TypeScript support

Get up and running with our framework-agnostic npm libraries, designed to be open-ended and work seamlessly with any peer technologies, front-end or back-end. 

npm install @callofduty/api
 
npm install @callofduty/assets

Get ready for rapid prototyping

We've simplified the nuances of authenticating Activision's various data sources to ensure you can quickly interact with every aspect of functionality. Fetch data, interact with friends list, block cheaters, and much more.

Need some inspiration?

Stagg.co is an open-source enhacement platform built on-top of this API. Functionality includes real-time data, auto-blocking cheaters, and more.
